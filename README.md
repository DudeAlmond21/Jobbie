## Overview
Jobbie is an iOS frontend application built using SwiftUI that connects college students
with nearby one-time jobs to help them manage daily expenses such as food, laundry, and
small services.

## Problem Statement
Many college students look for short-duration jobs to support basic daily needs.
Jobbie provides a simple mobile interface where students can discover nearby one-time
job opportunities like lawn mowing, painting, or library work.

## Features
- Startup animation and onboarding flow
- Home feed displaying available one-time jobs
- Search functionality
- Job detail screen with description and location
- MapKit integration to show job location
- Login and profile UI screens

## Tech Stack
- Swift
- SwiftUI
- MapKit
- MVVM-style architecture

## Project Status
This project currently focuses on frontend UI development.
Data is hardcoded for demonstration purposes.
Backend integration using Firebase for authentication and job data
is planned as a future enhancement.

## Architecture
The app follows a modular SwiftUI structure:
- Views for UI rendering
- Models for job data
- ViewModels for state handling and logic
